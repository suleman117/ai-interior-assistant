# AI Interior Assistant

## Overview
This project helps users generate interior design ideas based on room type and budget.

## Features
- Suggests design style
- Provides furniture ideas
- Recommends color combinations
- Gives estimated cost

## Tech Stack
- Java
- Spring Boot
- REST API

## Status
Project in progress (AI integration coming soon)

## Example Input
2BHK hall, budget 2 lakh

## Example Output
- Style: Modern
- Furniture: Sofa, TV Unit
- Color: White and Grey
- Estimated Cost: ₹2 lakh
